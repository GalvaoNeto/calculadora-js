# Calculadora JavaScript ✨

Uma calculadora moderna feita com HTML, CSS e JavaScript.

## 📌 Funcionalidades
- Operações matemáticas básicas (+, -, ×, ÷)
- Modo Claro/Escuro 🌞🌙
- Histórico de cálculos 🔢
- Suporte ao teclado ⌨️
- Botão para copiar resultado 📋

- ## 🛠️ Tecnologias usadas
- HTML5
- CSS3
- JavaScript (Vanilla)
